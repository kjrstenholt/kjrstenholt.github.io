Repeating Patterns with Illustrator and Spoonflower

I think you will have fun with this assignment! Do these in order, because they all use the same Illustrator file, and build on each other. Full instructions are in the videos. You can use new graphics, or adapt your sticker graphics if you like. You can use vector or raster images; my examples show raster images (photographs) so that I can show you key things about resolution, etc.
